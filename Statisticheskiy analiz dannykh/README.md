# GoFast — Анализ поведения пользователей сервиса аренды самокатов

## Описание проекта

**GoFast** — сервис проката электросамокатов, предоставляющий пользователям возможность совершать краткосрочные поездки по городу с помощью мобильного приложения.

Сервис предлагает два формата использования:
- с подпиской;
- без подписки.

Наличие подписки влияет на стоимость поездок и потенциально формирует различные модели поведения пользователей, что отражается на ключевых бизнес-показателях сервиса.

---

## Цель исследования

Определить, как наличие подписки влияет на поведение пользователей сервиса GoFast и ключевые метрики поездок, а также выявить различия между пользователями с подпиской и без неё.

---

## Задачи исследования

В рамках проекта были выполнены следующие задачи:

1. Проведено первичное изучение структуры и качества данных.
2. Выполнена предобработка данных:
   - обработка пропущенных значений;
   - удаление дубликатов;
   - анализ и обработка аномалий.
3. Выполнен исследовательский анализ данных (EDA).
4. Проведено сравнение поведения пользователей с подпиской и без неё.
5. Проверены статистические гипотезы.
6. Сформулированы выводы и рекомендации для бизнеса.

---

## Описание данных

В исследовании использовались данные о поездках пользователей сервиса GoFast, включающие информацию о:

- пользователях;
- поездках;
- длительности и расстоянии поездок;
- стоимости поездок;

## Выводы

Пользователи с подпиской демонстрируют более высокую активность в рамках отдельного месяца, что выражается в большей продолжительности поездок и более высокой помесячной выручке. Это делает подписку эффективным инструментом повышения краткосрочной ценности пользователя.

В то же время анализ показал, что **средняя годовая выручка на одного пользователя без подписки выше**, чем у пользователей с подпиской. Вероятной причиной является более длительный период активности пользователей без подписки и большее количество месяцев использования сервиса.

Таким образом, подписка увеличивает выручку в расчёте на активный месяц, однако не гарантирует более высокую годовую выручку на одного клиента. Для роста общей выручки компании целесообразно дополнительно исследовать:
- удержание пользователей с подпиской;
- причины оттока подписчиков;
- механизмы увеличения срока использования сервиса подписчиками.
